# Setup Gatsby with React Leaflet

[React Leaflet](https://react-leaflet.js.org/) is a component library bringing React Components to [Leaflet](https://leafletjs.com/).

---

## Install NPM packages

<NPMPackage name="leaflet" />
<NPMPackage name="react-leaflet" />
<NPMPackage name="gatsby-plugin-react-leaflet" />

---

## Install the React Leaflet plugin in gatsby-config.js

<GatsbyPlugin name="gatsby-plugin-react-leaflet" />
